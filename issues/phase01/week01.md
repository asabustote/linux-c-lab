iGit initはGitプロジェクトを生成するためのコマンド。
GitHub上にリポジトリを生成するには下記コマンドが必要
```
gh repo create asabustote/linux-c-lab --public --source=. --remote=origin
```

作成したスクリプトに実行権限を付与するコマンド
```
chmod +x somescript.sh
```
