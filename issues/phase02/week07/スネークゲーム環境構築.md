UbuntuでC言語のスネークゲームを作る準備を最短でシンプルにまとめるね。
これだけ整えればすぐコーディングに入れるよ。

⸻

✅ 1. 開発に必要なパッケージを入れる

GCC（コンパイラ）と make

sudo apt update
sudo apt install -y build-essential

👉 gcc と make が入る。

⸻

✅ 2. 画面制御ライブラリ（ncurses）を入れる

スネークゲームのように
	•	キー入力をリアルタイムで取得
	•	画面の一部を書き換える
	•	カーソルを消す

などをする時は ncurses が定番。

ncurses をインストール

sudo apt install -y libncurses5-dev libncursesw5-dev


⸻

✅ 3. プロジェクトフォルダを作る

例として snake-game ディレクトリを作る。

mkdir ~/snake-game
cd ~/snake-game


⸻

✅ 4. 最小の動作テスト（ncursesが動くか確認）

main.c を作成👇

#include <ncurses.h>

int main() {
    initscr();            // ncurses開始
    noecho();             // 入力文字を表示しない
    cbreak();             // 1文字入力即時読み取り
    keypad(stdscr, TRUE); // 矢印キー対応

    mvprintw(5, 10, "Hello ncurses!");
    refresh();

    getch();              // 何か押すまで待つ
    endwin();             // ncurses終了

    return 0;
}

コンパイル👇

gcc main.c -o test -lncurses

実行👇

./test

Hello ncurses! と表示されれば準備OK！

⸻

✅ 5. スネークゲームの基本構成

スネークは以下を作れば完成する：

① ゲームボード
	•	画面の枠を描画
	•	中にスネークと餌を表示

② スネークのデータ構造
	•	スネークの体（座標の配列またはリスト）
	•	進行方向（上/下/左/右）

③ 入力処理
	•	getch() と nodelay(stdscr, TRUE) を使う
	•	毎フレームキー入力をチェック

④ ゲームループ
	•	スネークを動かす
	•	壁 or 自分にぶつかったらゲームオーバー
	•	餌を食べたらスネークを伸ばす & 新しい餌生成
	•	画面を再描画
