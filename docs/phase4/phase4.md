# Phase 4：電子工作（8週）

## 概要
Raspberry Pi / ESP32を用いてGPIO・センサー制御、データ取得、自動制御を学ぶ。

## 週次計画

| Week | 内容 | 平日下調べ例 | 土日集中作業 |
|------|------|---------------|---------------|
| 1 (Design Week) | 回路・制御設計 | GPIO, I2C, SPI, センサー仕様確認 | 回路図作成、Raspberry Pi/ESP32初期セットアップ |
| 2 | LED/センサー制御 | pigpio/WiringPi, ADC/DAC | LED点灯、センサー値取得テスト |
| 3 | データ取得・ログ管理 | ファイル入出力、時間管理、CSV記録 | データログ記録プログラム作成 |
| 4 | 自動水やり/天気ステーション制御 | GPIOフロー制御、制御ループ | ポンプ制御、データ連動ロジック |
| 5 | systemd常駐化 | ユニットファイル作成、ログ保存方法 | プログラム常駐化、デバッグ |
| 6 | センサー統合・調整 | 複数センサー統合、閾値設定 | 複合制御テスト、安定化 |
| 7 | 安定動作確認 | エラー処理、例外管理、センサーキャリブレーション | 長時間運転テスト |
| 8 (Blog Week) | 振り返り・技術ブログ作成 | - | フェーズ全体まとめ記事作成、回路図・コード整理 |
