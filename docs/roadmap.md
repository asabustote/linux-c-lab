# 🗺️ Linux-C-Lab ロードマップ（全体像）

## 目的  
 Linux・C・ネットワーク・IoT基盤を自在に扱い、自作サーバ・ルータ・電子工作までの実践力を身につける。  
 将来的には自作OSやドライバ開発も視野に入れた低レイヤ技術力を習得する。

---

## 📌 プロジェクトゴール
- 「10年キャリア × CS学士」相当の技術力と知識を身につける  
- 自作サーバ・ルータ・IoTシステム構築  
- 将来的に自作OS・ドライバ・モジュール開発に挑戦  
- オンプレ環境中心、自作主義

## 📌 全体フロー

1. **Phase1 → Phase2 → Phase3 → Phase4 → Phase5**  
   各Phaseは「Design Week」で設計、「Blog Week」で振り返りと記録を実施  
2. 平日は設計メモ・調査、土日は集中作業（設計＋実装＋テスト＋ブログ作成）  
3. Phase6は長期目標としてREADMEで進捗・学びを記録  
4. 各Phase詳細はリンク先のファイルで週ごとのタスク・チェックリストを確認可能

| 5 | Mailサーバ構築 | - Postfix/Dovecot導入<br>- SMTP/IMAP動作確認<br>- メール送受信テスト | メールログ、設定ファイル |
| 6 | **Blog Week** | - Phase1まとめ記事作成<br>- 注意点・Tips記録 | 技術ブログ記事、GitHub更新 |

**平日作業例**  
- 設定ファイルの調査・テンプレート作成
